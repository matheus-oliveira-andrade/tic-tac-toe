### Tic-tac-toe
Implementation of tic-tac-toe in Go with interaction by terminal

### Features
- Play against another human player.
- Play against the computer, which uses the Minimax algorithm for optimal moves.

### How to run
```bash
make run 
# OR
# go run cmd/main.go 
```

```bash
make test
# OR
# go test ./... -v 
```
